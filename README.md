# Isorion Official Website

Official website for Isorion - Universal Infrastructure Compiler

**Live Site:** https://isorion.org

## Structure

```
isorion-website/
├── index.html    # Homepage
├── style.css     # Styling
└── README.md     # This file
```

## Deploy to GitHub Pages

This site is deployed via GitHub Pages from repository: `isorion-lang/isorion-lang.github.io`

### Setup

```bash
# Clone repo
git clone https://github.com/isorion-lang/isorion-lang.github.io.git
cd isorion-lang.github.io

# Copy files
cp ../isorion-website/* .

# Push
git add .
git commit -m "Add professional homepage"
git push origin main
```

Site will be live at https://isorion.org within 1-2 minutes.

## Local Development

Open `index.html` in browser to preview locally.
